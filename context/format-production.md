# Format & production system

## Format (operator-confirmed)
**Faceless animated diagrams + AI voiceover.** ByteByteGo-style progressive architecture diagrams, but ELI5 in tone. No face, no self-recorded voice (hard constraints).

## Visual identity (decide once, never change)
- Fixed template: same background color, same 2–3 accent colors, same font, same diagram style every reel → instant recognition in feed.
- 9:16 vertical, big centered text hook frame 1, diagram builds element-by-element (a new box/arrow appears every 1–2s so there is always motion).
- Burned-in captions always (≈50% muted viewing).

## $0 toolchain
- **Diagrams/animation**: Canva Free (animated elements, 9:16 export) or Excalidraw screen-captured while drawing; Keynote/PowerPoint Morph transitions exported as video also works. (Motion Canvas / Manim are free but higher learning curve — optional upgrade later.)
- **AI voice (critical dependency)**: CapCut built-in AI voices (free, no watermark, inside the editor — primary option); ElevenLabs free tier (~10 min/mo — best quality, use for hooks); Play.ht / LOVO / Murf free tiers as backups. ~16 reels/month × 45s ≈ 12 min of voice/month — CapCut free is the only tier that comfortably covers it. Verify commercial-use terms of whichever voice is used.
- **Editing/captions**: CapCut (free) — assemble diagram video + AI voice + auto-captions + trending instrumental audio.
- **Scheduling/analytics**: Meta Business Suite (free).

## Weekly workflow (fits 5–8 hrs)
1. **Sun (2–3h)**: pick 4 topics from backlog, write 4 scripts (60–90 words each: hook → analogy → diagram walkthrough → interview beat → CTA "save this / send to a friend prepping").
2. **Sun (2h)**: build all 4 diagram animations in one Canva session (template reuse).
3. **Weeknights (2×1h)**: CapCut assembly — voice, captions, audio; write captions/hashtags; schedule Mon/Wed/Fri/Sat posts.
4. **15 min Fri**: review retention + shares data of the week's posts; note what to change.

## Per-reel spec
- Length: 30–60s target (retention > length).
- Hook text ≤8 words, frame 1, no intro.
- One concept per reel. End with a send/save CTA.
- Caption: restate concept in 2 lines + 3–5 niche hashtags (#systemdesign #techinterview #softwareengineering #codinginterview).
