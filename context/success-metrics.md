# Success metrics & pivot triggers

## Primary metric (operator-confirmed)
**Follower count. 50k by month 9 is THE goal.** Operator explicitly wants aggressive format pivots if behind. Engagement/DMs/inbound are diagnostics, not goals.

## Milestone curve (growth is back-loaded; benchmarks in platform-playbook.md)
| Month | On-track followers | Notes |
|---|---|---|
| 1 | 300–1k | Slowest phase; judge retention %, not followers |
| 3 | 2k–4k | Should have ≥1 reel >100k views by now |
| 5 | 6k–12k | Format iteration mostly settled |
| 7 | 15k–25k | Needs 1–2 breakouts (>500k views) |
| 9 | 50k | Requires compounding + 2–4 breakout reels total |

## Weekly diagnostics (leading indicators)
- 3-second hold rate ≥60% (below 40% = hook problem → rewrite hooks first).
- Average watch % ≥70% on <45s reels (low = pacing/length problem).
- Sends-per-reach — the #1 weighted signal in 2026; if saves/sends are low, content isn't "useful enough to forward".
- Follows per 1k reach ≥5 (low = profile/positioning problem, fix bio & pinned reels).

## Pivot triggers (aggressive, per operator preference)
- **2 consecutive weeks** of hold rate <40% → new hook formulas.
- **Month 3 with <1.5k followers** → change visual format (e.g., switch Canva animation → screen-drawn Excalidraw style, or test text+b-roll), keep pillars.
- **Month 5 with <5k** → bigger swing: test carousel-heavy mix, collab/remix with adjacent accounts, or reposition depth (ELI5 → interview-answer-focused).
- **Any reel >250k views** → make 3 immediate follow-ups in the same sub-format (algorithm momentum window).
- Never pivot the niche itself (system design for preppers) before month 6 — niches need time to compound.

## Non-negotiables that don't pivot
4 reels/week cadence; one-concept-per-reel; technical correctness.
