# Goal

Launch an Instagram channel about high-level system design for computer science students. Reach **50,000 followers in 9 months** (by ~April 2027, starting July 2026), posting **4 reels/week** (~156 reels total).

## Operational interpretation
- 50k followers is THE metric. The operator wants formats pivoted aggressively if growth lags (see success-metrics.md).
- "High-level system design" = architecture concepts (caching, load balancing, sharding, queues, "design X" walkthroughs) — not low-level code.
- 4 reels/week is the floor commitment; carousels may be added as a secondary format if data supports it (see platform-playbook.md).
- Channel starts from zero: no handle, no existing audience, no backlog (see operator.md).
